# Why Emp v2.0

## ⚡️版本定位
+ 系统化解决现代web开发的耦合度问题
+ 实现真正意义上的微前端、微组件化
+ 打造完整生态实现从开发到部署的一体化解决方案
+ 更加完善的配置与调优

## 📦特有功能
+ 实现 `empshare` 3级共享模型
+ 重写 `d.ts` 生成于同步逻辑，实现跨业务开发的自动同步体验
+ 基于 `swc` 深度定制 `js`与`ts` loader，实现更加高效的开发体验
